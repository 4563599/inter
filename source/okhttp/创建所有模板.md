## 💡 每个模板的特色

### 1. 详细的类注释
```java
/**
 * XXX 类
 * 
 * 【为什么需要这个类？】
 * - 原因1
 * - 原因2
 * 
 * 【为什么这样设计？】
 * - 设计原因1
 * - 设计原因2
 */
```

### 2. 详细的字段注释
```java
/**
 * 字段名
 * 
 * 【为什么需要这个字段？】
 * - 作用说明
 * 
 * 【为什么是 final？】
 * - 不可变对象
 */
private final String field;
```

### 3. 详细的方法注释
```java
/**
 * 方法名
 * 
 * 【为什么需要这个方法？】
 * - 方法作用
 * 
 * 【实现思路】
 * 1. 步骤1
 * 2. 步骤2
 * 
 * 【注意事项】
 * - 注意点1
 * - 注意点2
 */
```

### 4. 清晰的 TODO 提示
```java
// TODO: 实现这个方法
// 步骤：
// 1. 做什么
// 2. 做什么
// 3. 注意什么
```

### 5. 完整的测试方法
```java

应该输出什么...
```
